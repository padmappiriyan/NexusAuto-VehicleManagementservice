# AutoMobile Service Management System

This Service is a **Vehicle Management and Task Update System** built to streamline service center operations.  
It allows admins and employees to manage vehicle service bookings, track progress, and update task statuses efficiently in real time.

###  Employee Side
- View only **In-Progress** tasks assigned to them.  
- Update task status to **Completed** once the job is done.  
- View detailed vehicle and service information for each task.

###  Customer Side
- Add a vehicle for service.  
- View booking status (Pending, Accepted, In Progress, Completed, or Rejected).  
- Receive updates when service is finished.
